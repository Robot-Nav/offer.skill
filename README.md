# offer.skill

**智能面试模拟器** — 基于简历的AI驱动面试准备工具

##  项目简介

**offer.skill** 是一款模拟真实面试场景的智能工具。你只需上传简历（支持 PDF、DOCX、Excel 等格式），系统便会自动解析简历内容，根据你的项目经验、技术栈、实习经历等关键信息，智能生成高度相关的面试问题与八股文知识点。通过直观的 HTML 问答交互界面，你可以获得一对一面试官的对话体验，随时随地进行面试模拟，高效备战求职。

##  核心功能

- **多格式简历解析**  
  支持上传 PDF、DOCX、Excel 等多种格式，自动提取简历中的项目、技能、实习等关键信息。

- **智能面试题生成**  
  基于简历内容，AI 自动生成与你经历强相关的个性化面试问题，避免千篇一律的泛泛之问。

- **八股文知识匹配**  
  根据你的技术栈，智能匹配对应的八股文高频考点，帮你巩固计算机基础、框架原理等核心知识。

- **一对一对话体验**  
  HTML 界面模拟真实面试官对话流程，支持多轮问答，沉浸式还原面试实感。

- **回答评估反馈**  
  对你的回答进行智能分析，指出亮点与不足，给出针对性改进建议，帮助你持续进步。
